# JS  基本语法

## 什么是表达式和语句
表达式是由操作数和运算符组成的式子。表达式是指具有完整意义的计算机指令。
## 标识符的规则
可以由字母 数字 符号 下划线等组成，不能用数字开头，不能用系统自带的单词作为标识符 比如 if  window 等
## if else 语句

 ``` 
 }if else() {

 }          
 ```
